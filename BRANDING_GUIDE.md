# EduSeek Branding Guide

## Business Information

- **Platform Name**: EduSeek
- **Full Name**: EduSeek - School Discovery Platform
- **Location**: South Africa
- **Domain**: eduseek.co.za
- **Website**: https://www.eduseek.co.za (when deployed)

## Logo Usage

### Logo Variants

1. **Full Logo** (`eduseek-logo-full.svg`)
   - Icon + Wordmark combination
   - Use in: Authentication pages, email headers, large branding areas
   - Minimum size: 200x60px

2. **Wordmark** (`eduseek-logo-wordmark.svg`)
   - Text "EduSeek" only
   - Use in: Navigation bar, footer, smaller branding areas
   - Minimum size: 120x24px

3. **Icon** (`eduseek-logo-icon.svg`)
   - ES monogram only
   - Use in: Favicon, app icons, small branding elements
   - Minimum size: 32x32px (square)

### Logo Colors

- **Gradient**: Blue → Teal → Green
- **Blue**: Primary brand color (#2563EB or similar)
- **Teal**: Mid-gradient color
- **Green**: Secondary brand color (#28a745 or similar)

### Logo Placement

- **Navigation Bar**: Wordmark logo (left side)
- **Authentication Pages**: Full logo (centered, above form)
- **Dashboards**: Wordmark or icon in header
- **Footer**: Wordmark logo
- **Favicon**: Icon only
- **Email Templates**: Full logo in header

## Typography

- **Primary Font**: System fonts (Arial, Helvetica, sans-serif)
- **Brand Name**: Always use "EduSeek" (capital E, capital S)
- **Never use**: "EDS", "eds", "Edu Seek", or any variations

## Color Scheme

- **Primary**: Blue (#2563EB)
- **Secondary**: Green (#28a745)
- **Gradient**: Blue → Teal → Green (matching logo)
- **Neutrals**: Gray scale for text and backgrounds

## Geographic Context

- **Business Location**: South Africa

- **Language**: English (primary)

## Brand Voice

- Professional
- Helpful
- Trustworthy
- Modern
- Accessible

## Implementation Notes

- All references to the platform must use "EduSeek"
- Logo files should be placed in `/public/logo/` directory
- Use the Logo component (`components/ui/Logo.tsx`) for consistent rendering
- Fallback to gradient text logo if image files are missing
- Maintain brand consistency across all user interfaces

## File Naming Convention

Logo files must be named exactly:
- `eduseek-logo-full.svg` (or .png)
- `eduseek-logo-wordmark.svg` (or .png)
- `eduseek-logo-icon.svg` (or .png)
